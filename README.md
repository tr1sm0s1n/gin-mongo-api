## ‚öôÔ∏è Run Locally

Clone the project

```bash
git clone https://github.com/tr1sm0s1n/gin-mongo-api.git
cd gin-mongo-api
```

Start the database

```bash
make up
```

View the database (optional)

```bash
make enter
```

Run the application

```bash
make run
```

For live reload, install air (optional)

```bash
make air
```

Run the application in watch mode

```bash
make air
```
